  • Common Mistakes

1.) Put RAM in slots on motherboard

Take motherboard out of box, set on a solid surface, put RAM sticks in RAM slots. Ensure RAM is seated correctly by listening for a click noise.

  Reference: Here!
 

2.) Install CPU to Motherboard

Follow reference link to install CPU/cooler into motherboard.

  Reference: Here!
 

3.) Install storage onto Motherboard

Install SDD and HDD's to motherboard or slots in case

  Reference: Here!
 

4.) Put GPU in its PCI slot

Put the GPU into the first PCI slot on the motherboard. It should be the slot closest to the CPU.

  Reference: Here!
 

5.) Ensure that the PC Posts

You should always make sure the PC POST's before installing everything into the case. A PC that doesnt POST equals faulty parts.

  Reference: Here!
 

6.) Install Fans into case

Install intake and outtake fans into the PC case.

  Reference: Here!
 

7.) Put motherboard with CPU/RAM into case

After the PC posts and you have fans installed, put the motherboard in the case

  Reference: Here!
 

8.) Connect all fans to fan headers

All motherboards only have a certain amount of fan headers. Some fans may need to be daisy chained.

  Reference: Here!
